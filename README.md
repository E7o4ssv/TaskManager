# DanyWhite — Менеджер компании

Внутреннее приложение для команды: общий чат, обмен файлами и контроль задач.

## Возможности

- **Авторизация** — регистрация и вход по email/паролю
- **Чат** — общий канал для переписки (обновление каждые 3 сек)
- **Файлы** — загрузка и скачивание файлов, список с автором и датой
- **Задачи** — создание задач, статусы (к выполнению / в работе / выполнено), приоритет, срок, назначение исполнителя

## Запуск

1. Установите зависимости:
   ```bash
   npm install
   ```

2. Создайте файл `.env` (можно скопировать из `.env.example`):
   ```
   DATABASE_URL="file:./dev.db"
   ```

3. Примените схему БД:
   ```bash
   npx prisma generate
   npx prisma db push
   ```

4. Запустите сервер:
   ```bash
   npm run dev
   ```

5. Откройте [http://localhost:3000](http://localhost:3000), зарегистрируйтесь и войдите.
